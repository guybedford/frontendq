If you want to use a dynamic import for angular in the Bootstrap instead of in the static module tree, then your bootstrap method should call back on the import promise.

Typically you want to always use the static tree (`import` and `export` syntax) because that will allow your modules to build into a bundle.

The use case for a `System.import` in the bootstrap is usually for a router where you want to load the page you're navigating to only when you navigate
